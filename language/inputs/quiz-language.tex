\quizAufgabe{Each logic program without choice rules has at most one stable model.}
\quizAufgabe{The stable models of a logic program $P\cup C$, where $C$ is a set of integrity

\quizAufgabe{Each logic program without integrity constraints has some stable model.}
\quizAufgabe{ASP systems first evaluate choice rules, and then prune

\end{quiz}
